【bash解释器常用的快捷键】
Tab→补齐命令，按2下列出包含的命令、选项。由bash-completion软件包提供该功能。
Ctrl L → 清除屏幕。
Ctrl U → 清除命令行。
Ctrl W → 清除命令行前一个单词(命令)。
Ctrl A → 光标移至命令行行首。
Ctrl E → 光标移至命令行行尾。
Ctrl C → 终止当前执行的程序。
Ctrl R → 反向搜索执行过的命令
Ctrl Alt T → 打开终端
Ctrl Shift T → 增加终端并列窗口
Ctrl + → 放大终端字体
Ctrl - → 缩小终端字体
